<h2 id="desc"> 📖 Descrição Geral</h2>

Este repositório contém um projeto de CRUD (Create, Read, Update, Delete) desenvolvido em Python, com o objetivo de gerenciar um estoque de produtos. 
O sistema permite realizar operações de cadastro, consulta, atualização e exclusão de itens, integrando um banco de dados Oracle.


<h2 id="func"> 🚀 Funcionalidadesl</h2>

### 🔹 Cadastrar Produtos
- Permite ao usuário adicionar novos produtos ao estoque, preenchendo informações como nome, descrição, e informções de financeiras.

### 🔹 Alterar Produtos
- Facilita a atualização das informações dos produtos já cadastrados no sistema, permitindo alterações em campos como nome, quantidade e preço.

### 🔹 Excluir Produtos
- Permite que o usuário exclua produtos do estoque, removendo-os permanentemente do banco de dados. O sistema realiza uma verificação para garantir que a exclusão seja feita de forma segura.

### 🔹 Mostrar Produtos
- Exibe uma lista completa dos produtos cadastrados no estoque, incluindo detalhes como nome, descrição, e as informações financeiras.

### 🔒 Segurança
- A descrição dos produtos no banco de dados é protegida por criptografia utilizando o algoritmo de Hill, garantindo maior segurança na manipulação dos dados.
